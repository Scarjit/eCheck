# eCheck

Simple utility to check if which of your apps will run on /e/ OS

# Requirements
 - Python3
 - [ADB](https://developer.android.com/studio/command-line/adb) in Path
# Installation
1. Download the repo
2. Plugin your phone and allow ADB access from your PC
3. Execute main.py and wait

## device_name_found.log
These apps can be installed on /e/

## device_name_missing.log
These apps are missing from the /e/ app store

## device_name_skipped.log
These apps are android & lineage internal and shouldn't require manual installation
